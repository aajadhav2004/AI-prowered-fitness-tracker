Intelli-Fit

Intelli-Fit is a fitness tracking system built to help users monitor their daily health and workout routines in one place. The application focuses on making fitness tracking simple, visual, and engaging, so users can easily understand their progress over time.

The platform combines activity tracking with helpful content like blogs and tutorials to encourage consistency and healthy habits.

Purpose of the Project

I built Intelli-Fit to learn how a full-stack application works from end to end — from managing user data on the backend to presenting progress visually on the frontend. The goal was to create a system that not only tracks data but also keeps users motivated.

Key Features

Daily calorie tracking

Workout logging for different routines

Weekly progress charts for better visualization

Built-in BMI calculator

Blogs and fitness tutorials to improve user engagement

Tech Stack

Frontend: React, Tailwind CSS

Backend: Node.js, Express.js

Database: MongoDB

What I Learned

Working on Intelli-Fit improved my understanding of full-stack development, REST APIs, and data visualization. I also learned how to design features that increase user engagement, resulting in higher user activity.
